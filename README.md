# Google Ads Month-over-Month Anomaly Detector & Dashboard Builder

This Google Ads script acts as an automated early warning system for your accounts. It compares the last fully completed month against the prior month to detect significant performance shifts (default 15%+) across Cost, CTR, Avg CPC, and specific Conversion Actions.

Unlike standard scripts that dump raw data, this script automatically builds a visual dashboard directly inside a Google Sheet (complete with trend charts and recommended next steps) and emails a summary report to your team.
